# Customer Behavior Analysis and Business Insights Dashboard

## Project Overview

This project focuses on analyzing customer purchasing behavior using Python, PostgreSQL, and Power BI. The objective is to extract meaningful business insights from customer data through data cleaning, exploratory data analysis (EDA), SQL-based analysis, and interactive dashboard creation.

The project follows a complete data analytics workflow, starting from raw data processing to business intelligence reporting.

---

## Objectives

* Understand customer purchasing patterns and behavior.
* Clean and prepare raw data for analysis.
* Perform exploratory data analysis (EDA) to identify trends and anomalies.
* Use SQL queries to answer business-related questions.
* Build an interactive Power BI dashboard for data-driven decision making.

## Project Workflow

### 1. Data Loading

* Imported customer dataset using Python.
* Examined dataset structure and data types.
* Checked for missing values and inconsistencies.

### 2. Data Cleaning

* Handled missing and duplicate records.
* Standardized column formats.
* Created new features where required.
* Prepared a clean dataset for further analysis.

### 3. Exploratory Data Analysis (EDA)

Performed analysis to understand:

* Customer demographics
* Purchase behavior
* Spending patterns
* Revenue distribution
* Frequency of purchases
* Customer segmentation trends

Visualizations were created to support findings and identify key business insights.

### 4. SQL Analysis (PostgreSQL)

The cleaned dataset was loaded into PostgreSQL and analyzed using SQL queries.

Example business questions addressed:

* Which customer segments generate the highest revenue?
* What are the most common purchasing patterns?
* How does spending vary across age groups?
* Which categories contribute most to sales?
* What factors influence customer loyalty?

### 5. Power BI Dashboard

An interactive dashboard was developed to visualize key metrics and insights.

Dashboard features include:

* Revenue overview
* Customer segmentation analysis
* Purchase frequency analysis
* Demographic insights
* Interactive filters and slicers
* Business KPI tracking

---

## Key Insights

* Identified high-value customer segments.
* Analyzed revenue contribution across different customer groups.
* Discovered purchasing trends and behavioral patterns.
* Generated actionable insights to support business decision-making.

## How to Run the Project

### Python Analysis

1. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

2. Open and run:

   ```bash
   eda.ipynb
   ```

### PostgreSQL Analysis

1. Create a PostgreSQL database.
2. Import the cleaned dataset.
3. Execute the SQL queries from:

   ```sql
   analysis.sql
   ```

### Power BI Dashboard

1. Open:

   ```
   customer_behaviour_dashboard.pbix
   ```
2. Refresh the data connection if required.
